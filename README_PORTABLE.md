# Orion Studio Portable Quickstart

Run from the `orion-raytracer/` directory:

Install: `python scripts/orion_portable.py install`

Run: `python scripts/orion_portable.py run`

Native wrappers:

- Windows: `.\install.bat` then `.\run.bat`
- Linux/macOS: `./scripts/install.sh` then `./scripts/run.sh`
